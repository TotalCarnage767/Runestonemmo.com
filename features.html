RunestoneCore

The Engine That Powers Every System, Every Module, Every Adventure.

Overview

RunestoneCore is the foundational module of the RunestoneMMO ecosystem — a modular, extensible, enterprise‑grade engine providing all essential MMO infrastructure: economy, moderation, menus, scoreboards, cooldowns, attributes, world rules, storage, logging, and developer APIs. Every other RunestoneMMO module plugs into Core’s lifecycle, registries, and APIs.

Unified Configuration & Storage

  • Single unified config.yml controlling all systems
  • YAML or SQLite storage
  • Auto‑save & backup intervals
  • Data compression
  • Performance monitoring
  • Debug logging
  • Centralized storage abstraction layer

RPG Systems Not Included (Yet)

Based on the IDE audit, the following systems are not implemented in Core and will be added in future modules:

Missing Combat Systems

  • No custom damage engine
  • No RPG combat math
  • No critical hits or armor formulas
  • No combat feedback

Missing Progression Systems

  • No leveling system
  • No skill points
  • No classes
  • No abilities

Missing Content Systems

  • No quest framework
  • No custom items
  • No item rarities
  • No item stat system

Missing Social Systems

  • No parties
  • No clans/guilds
  • No chat channels

Missing Persistence

  • Attributes not saved persistently
  • Cooldowns not saved persistently
  • No MySQL/PostgreSQL support

Incomplete Features

  • Auction search/filter
  • Player shop item detail view
  • Attribute save/restore methods
