Runestone Vale Features & Commands

A Paper 1.21.11 fantasy MMORPG survival realm built around progression, commandless utility, quests, economy, custom content, and long-term world identity.

This page is the detailed feature and command guide for Runestone Vale. It covers the current live systems, player-facing tools, server progression, staff-supported systems, and the most important commands players or staff may use.

Runestone Vale is powered by the custom RunestoneMMO plugin suite and supported by trusted tools for custom assets, NPCs, permissions, world protection, economy compatibility, Bedrock access, and build support.

Server at a Glance

Runestone Vale is a fantasy MMORPG survival server with medieval, dwarven, and rune-forged identity. It is being built around long-term progression, economy, quests, NPCs, custom content, exploration, world control, and systems that are meant to feel connected instead of scattered.

Platform

Plugin Stack Basics

The custom RunestoneMMO modules are the main identity of the realm. Supporting plugins provide permissions, NPCs, custom assets, world protection, build tooling, cross-platform access, and compatibility.

RunestoneMMO Plugins

  • RunestoneCore — shared server foundation, economy, menus, travel, world rules, scoreboards, claims, graves, items, mail, and core APIs
  • RunestoneSkills — skills, professions, XP, gathering, crafting direction, Oraxen skill assets, and skill progression
  • RunestoneCombat — custom combat framework, attributes, scaling, damage direction, and combat progression
  • RunestoneStamina — stamina state, movement pacing, survival pressure, stamina HUD, buffs, and stamina brews
  • RunestoneHub — Login Hub, Hub Compass, travel access, market access, bank access, quest info, and updates
  • RunestoneNPCs — NPC bindings, dialogue runtime, quest runtime, NPC shops, and world interaction systems
  • RunestoneAdminTool — Admin Scepter, Admin Control Nexus, staff menus, debug tools, and controlled world access

Supporting Server Plugins

  • Oraxen — custom items, blocks, models, textures, and resource-pack identity
  • Citizens — NPC framework for characters, quest givers, shops, and interaction points
  • Sentinel — NPC combat/security support where needed
  • Denizen — scripting support where used for world interaction
  • WorldGuard — protected regions and controlled areas
  • WorldEdit — building, schematic work, and world shaping
  • LuckPerms — ranks, permissions, and staff access control
  • Vault — economy compatibility layer
  • PlaceholderAPI — placeholders for displays, scoreboards, and integrations
  • Geyser-Spigot + Floodgate — Bedrock access support
  • ViaVersion / ViaBackwards / ViaRewind — version compatibility support
  • Discord-Linker — Discord/server connection support
  • packetevents — packet-level support used by compatible systems
